# GASバックエンド接続手順（データ共有・認証強化）

事務管理システムを「この端末だけ（localStorage）」から「全端末で共有（Googleスプレッドシート）」に切り替える手順です。
会員管理システムとは**別の**スプレッドシート・別のGASプロジェクトを使います（データは混ざりません）。

所要時間: 15〜20分ほど。順番に進めてください。

---

## 1. スプレッドシートを用意する

1. https://sheets.google.com を開き、空のスプレッドシートを新規作成
2. 名前を「たかすスポーツクラブ 事務管理データ」など分かる名前にする
3. URLの中の **スプレッドシートID** を控える
   例: `https://docs.google.com/spreadsheets/d/`**`ここがID`**`/edit`

## 2. GASプロジェクトを作成しコードを貼る

1. https://script.google.com を開き「新しいプロジェクト」
2. 左の `コード.gs` の中身をすべて消し、この repo の [`gas/Code.gs`](Code.gs) の中身を全部貼り付けて保存
3. プロジェクト名を「たかすスポーツクラブ 事務管理」にする

## 3. スクリプト プロパティを設定する

GASエディタ左の「⚙ プロジェクトの設定」→「スクリプト プロパティ」で、次の3つを追加：

| プロパティ | 値 |
|---|---|
| `SPREADSHEET_ID` | 手順1で控えたID |
| `ADMIN_EMAIL` | 事務局のログイン用メールアドレス |
| `ADMIN_PASSWORD` | 事務局のログイン用パスワード（**推測されにくいものに**） |

※ `admin123` のような簡単なものは避けてください。これがログイン認証の要になります。

## 4. 初期化を実行する

1. GASエディタ上部の関数選択で **`setupSpreadsheet`** を選び「実行」
2. 初回は権限の承認を求められる → 自分のGoogleアカウントで許可
   （「このアプリは確認されていません」→「詳細」→「（プロジェクト名）に移動」で進む）
3. 実行ログに「セットアップ完了」と出ればOK。スプレッドシートに「管理者」「職員」「勤怠」「有給休暇」「シフト」のタブができます

## 5. ウェブアプリとしてデプロイする

1. GASエディタ右上「デプロイ」→「新しいデプロイ」
2. 種類の選択（⚙）で「**ウェブアプリ**」
3. 設定：
   - 説明: 任意
   - 次のユーザーとして実行: **自分**
   - アクセスできるユーザー: **全員**（アプリ独自のトークンで認可するため）
4. 「デプロイ」→ 権限承認
5. 表示された **ウェブアプリのURL**（`https://script.google.com/macros/s/…/exec`）をコピー

## 6. サイトにURLを設定する（GitHub）

1. https://github.com/takasucot2028-del/takasu-office/settings/variables/actions を開く
2. 「New repository variable」
   - Name: `VITE_GAS_URL`
   - Value: 手順5でコピーしたウェブアプリURL
3. 「Add variable」

## 7. 再デプロイして反映

- GitHubの Actions ページで最新のワークフローを「Re-run jobs」する
- （または私に「再デプロイして」と言っていただければ空コミットで反映します）
- 1〜2分後、サイトが「共有モード」になります

---

## 動作確認

- サイト https://takasucot2028-del.github.io/takasu-office/ を開く
- ログイン画面下の「デモアカウント」の表示が**消えていれば**共有モードに切り替わっています
- 手順3で設定した `ADMIN_EMAIL` / `ADMIN_PASSWORD` でログイン
- 職員を1名登録 → 別の端末（スマホ等）でログインして同じ職員が見えれば成功

## パスワードを変えたいとき

- ログイン後、画面上部「設定」→「パスワードの変更」から変更できます
- 忘れた場合は、GASエディタで `ADMIN_PASSWORD` を新しい値にしてから関数 `resetAdminCredentials` を実行してください

## コードを更新して再デプロイするとき（重要）

`gas/Code.gs` を修正したら、GASエディタで貼り替えたあと、
「デプロイ」→「デプロイを管理」→ 既存デプロイの ✏ 編集 → バージョン「新しいバージョン」→「デプロイ」
としてください（URLは変わりません）。「新しいデプロイ」を作るとURLが変わってしまうので注意。
